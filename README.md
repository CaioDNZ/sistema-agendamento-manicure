📅 Sistema de Agendamento para Manicure

Projeto desenvolvido com MySQL para praticar modelagem de banco de dados relacional, criação de tabelas, relacionamentos, inserção de dados e consultas SQL.
 O sistema simula o gerenciamento de clientes, serviços, agendamentos e pagamentos de uma manicure autônoma.

🎯 Objetivos
- Praticar modelagem relacional.
- Aplicar boas práticas na criação de tabelas.
- Utilizar chaves primárias e estrangeiras.
- Desenvolver consultas SQL do nível básico ao intermediário.

🛠 Tecnologias
- MySQL
- MySQL Workbench
